# ddnsc v1.0.0

Just another simple & lightweight client to update DNS dynamically. Written in python with easy way of supporting new service providers with REST API.

## Requirements

 - Python v3 or later
   - python-systemd
   - python-requests

|--------------|---------------------|

| DreamHost    | :red_circle:        |
